# P1 — AI-Powered Log Analysis Agent

A Claude-powered security agent that answers natural language questions about real SSH honeypot logs from an AWS EC2 server. Built as Project 1 of a 12-project cloud security portfolio.

---

## What It Does

Ask the agent security questions in plain English. It decides which tools to call, runs them against 86,839 real log events, and synthesizes a clear answer — streamed to your terminal in real time.

```
You: Was there a brute force attack?

Agent: Yes — 361 unique IPs exceeded the detection threshold across the
entire month of December 2024. Four IPs used the exact same 47-username
wordlist on different days, a clear sign of coordinated botnet activity...
```

---

## Dataset

**Source:** [SecRepo Security Log Samples](https://www.secrepo.com/) — `auth.log` from an AWS EC2 honeypot (`ip-172-31-27-153`, us-east-1)

| Metric | Value |
|---|---|
| Period | Nov 30 – Dec 31, 2024 |
| Total events | 86,839 lines |
| Invalid-user attempts | 12,250 |
| Brute-force cutoffs | 2,575 |
| Unique attacking IPs (≥5 attempts) | 361 |
| Port-scan probes | 969 |

The server rejects attackers at **pre-auth** — there are no "Failed password" events. All attack signals come from `Invalid user`, `Too many authentication failures`, and `Did not receive identification string` messages.

## Architecture

```
User question (natural language)
        │
        ▼
┌───────────────────────────────┐
│   Claude Opus 4.6             │  ← decides which tools to call
│   (streaming agentic loop)    │
└───────────────┬───────────────┘
                │  tool_use
        ┌───────┴────────┐
        ▼                ▼
  Tool execution    More tools
  (Python)         (parallel)
        │
        ▼
┌───────────────────────────────┐
│   auth_modern.log             │  ← 86,839 ISO 8601 events
│   (data/auth_modern.log)      │
└───────────────────────────────┘
        │
        ▼
Claude synthesizes tool results
        │
        ▼
Plain-English answer (streamed)
```

### Files

```
P1_log_analysis_agent/
├── data/
│   ├── auth.log              ← raw SecRepo dataset (legacy syslog timestamps)
│   └── auth_modern.log       ← modernized (ISO 8601, generated by modernize_logs.py)
├── src/
│   ├── modernize_logs.py     ← converts legacy timestamps to ISO 8601
│   ├── tools.py              ← four agent tools (search, count, detect, correlate)
│   ├── agent.py              ← Claude-powered agent with streaming agentic loop
│   └── regex.py              ← shared regex patterns
├── tests/
│   └── test_tools.py         ← 45 tests across all four tools
└── DECISIONS.md              ← design decision log
```

---

## Agent Tools

| Tool | Purpose |
|---|---|
| `search_logs(query)` | Search log lines for a keyword, IP, or username |
| `count_events(event_type)` | Count events by category (invalid_user, disconnect, etc.) |
| `detect_brute_force(threshold)` | Rank IPs by hostile attempts; return attack windows and targeted users |
| `correlate_events(ip)` | Full chronological timeline for a single IP |

## Setup

**Requirements:** Python 3.10+, an Anthropic API key

```bash
# 1. Clone
git clone https://github.com/thangdvo/P1-log-analysis-agent
cd P1-log-analysis-agent

# 2. Install dependencies
pip install anthropic python-dotenv pytest

# 3. Set your API key
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

# 4. Add the log data (not included in repo — too large)
# Download auth.log from https://www.secrepo.com/ and place in data/

# 5. Modernize the timestamps
python src/modernize_logs.py

# 6. Run the demo
python src/agent.py --demo

# 7. Or ask your own questions interactively
python src/agent.py
```
